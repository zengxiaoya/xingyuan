# 一、产品定位

## 1.1 一句话描述
一款让小学生通过「闯关 + 和 AI 聊天」探索太空知识的 Web 游戏。

## 1.2 目标用户
- 主要用户：6–10岁小学生（能独立操作，自主探索）
- 陪伴用户：家长（偶尔一起玩，不是必须参与）

## 1.3 核心价值
| 用户痛点 | 本产品解法 |
|---------|-----------|
| 太空知识枯燥难记 | 游戏闯关，答题才能解锁星球 |
| 百科是被动灌输 | AI 不给答案，引导孩子自己思考 |
| 问题没人解答 | NOVA 船长随时在线，追问不限次 |
| 学完就忘 | 徽章收集+证书系统，强化成就感 |

## 1.4 精神理念
> 光从深渊来  
> 知识像星光  
> 穿越亿万年的黑暗  
> 照亮此刻的你

# 三、关卡体系

## 3.1 解锁机制
```
第一阶段（默认全开）
├── 太阳系 Lv1：行星基本知识
├── 宇宙探索 Lv1：宇航员和火箭
└── 宇宙尺度 Lv1：星系和光年

↓ 三个 Lv1 全部通关 → 解锁第二阶段

第二阶段
├── 太阳系 Lv2：探测器和人类探索
├── 宇宙探索 Lv2：空间站和太空生活
└── 宇宙尺度 Lv2：黑洞和星云

↓ 三个 Lv2 全部通关 → 解锁第三阶段

第三阶段
├── 太阳系 Lv3：极端环境和生命可能性
├── 宇宙探索 Lv3：未来星际移民
└── 宇宙尺度 Lv3：宇宙大爆炸
```

## 3.2 关卡内容与解锁科学家

| 关卡 | 主题内容 | 解锁科学家 |
|------|---------|-----------|
| 太阳系 Lv1 | 行星基本知识（8大行星+太阳、冥王星、主要卫星） | 哥白尼、伽利略 |
| 太阳系 Lv2 | 探测器和人类探索（旅行者号、好奇心号等） | 卡尔·萨根、冯·布劳恩 |
| 太阳系 Lv3 | 极端环境和生命可能性（木卫二、土卫六、火星生命） | 弗兰克·德雷克 |
| 宇宙探索 Lv1 | 宇航员和火箭（航天史、火箭原理、著名宇航员） | 尤里·加加林、阿姆斯特朗 |
| 宇宙探索 Lv2 | 空间站和太空生活（国际空间站、中国空间站） | 王亚平 |
| 宇宙探索 Lv3 | 未来星际移民（火星移民计划、星舰） | 埃隆·马斯克 |
| 宇宙尺度 Lv1 | 星系和光年（银河系、宇宙大小、光速概念） | 哈勃 |
| 宇宙尺度 Lv2 | 黑洞和星云（黑洞形成、事件视界、第一张黑洞照片） | 霍金、Katie Bouman |
| 宇宙尺度 Lv3 | 宇宙大爆炸（宇宙起源、暗物质、宇宙未来） | 勒梅特 |

## 3.3 每关题型结构

| 题序 | 题型 | 规则 |
|------|------|------|
| 第1题 | 选择题 | 4选1，有明确对错，答错可重答最多2次 |
| 第2题 | 猜谜题 | AI出谜面，自然语言作答，AI语义判断，可申请2次提示 |
| 第3题 | 创意题 | 无标准答案，AI给个性化反馈，必定正向 |

每道题附带对应科学家的发现背景或名言。答题后 NOVA 补充延伸知识（不超过30字）。

## 3.4 通关奖励
- 解锁关卡徽章（存入成就页和证书）
- 解锁对应科学家故事卡片
- 创意题答案自动保存（可在成就页回看）
- 获得星星积分

# 四、AI 能力设计

## 4.1 NOVA 行为规则
1. 不直接给闯关题答案，用反问引导
2. 每 2–3 轮主动提问，引导孩子思考
3. 遇到好问题先夸奖再回答
4. 每次回复不超过 80 字，用生活比喻代替术语
5. 感知当前上下文（正在看哪个关卡/科学家）
6. 保留本次会话历史，多轮对话连贯

## 4.2 AI 负责 vs 不负责

| AI 负责 | 前端逻辑负责 |
|--------|------------|
| 实时生成题目 | 星球冷知识（写死，保证准确） |
| 评估猜谜题语义 | 积分计算、进度存储 |
| 创意题个性化反馈 | 徽章解锁条件 |
| NOVA 角色对话 | 页面路由、动画效果 |

## 4.3 数据存储

# 六、页面设计规范

## 6.1 登录开场序列

**形式**：全屏沉浸，共6幕约30秒，右上角可跳过。

| 幕次 | 时长 | 内容 | 动效 |
|------|------|------|------|
| 第1幕 | 0–1.2s | 纯黑宇宙，星粒浮现 | 星星粒子随机漂浮闪烁 |
| 第2幕 | 1.2–4.5s | 星系漩涡Logo旋转生长 | scaleIn+rotate，发光特效 |
| 第3幕 | 4.5–9.5s | 四句诗词逐行浮现 | 每行间隔0.7s fadeIn，highlight行放大 |
| 第4幕 | 9.5–15s | 制作背景文案淡入 | 每行间隔0.5s，关键词高亮紫色 |
| 第5幕 | 15s+ | 发光星球等待点击 | pulse动画，点击触发粒子爆散 |
| 第6幕 | 点击后 | 登录表单从深处升起 | riseUp cubic-bezier弹性动画 |

**诗词文案**：
> 光从深渊来 / 知识像星光 / 穿越亿万年的黑暗 / 照亮此刻的你

**制作背景文案**：
> 这是一个二年级小朋友和爸爸 / 一起探索宇宙的故事 / 用代码丈量星辰大海

**登录表单字段**：姓名 / 年级（下拉）/ 学校名称 / 头像选择（男宇航员×2、女宇航员×2）

---

## 6.2 星图总览

**布局**：星空地图风，9个关卡节点散布宇宙背景，三列虚线分区。

**分区**：太阳系（左）/ 宇宙探索（中）/ 宇宙尺度（右），横向虚线划分 Lv1/Lv2/Lv3 层次。

**节点状态**：
- 已通关：绿色✓，外层绿色呼吸光晕
- 可挑战：紫色亮点，旋转椭圆轨道+卫星动画
- 未解锁：深色🔒，虚线边框

**动画效果**：
- 节点浮动：每个节点以不同频率/幅度上下漂浮
- 连线流光：已解锁路径实线，3颗粒子持续流动；未解锁虚线
- 星云背景：三区域有极淡色调区分

**交互**：悬停显示 Tooltip（关卡名、科学家、状态），点击可进入关卡。

**固定元素**：右下角 NOVA 按钮（脉冲光晕），左下角状态图例，顶部进度条。

---

## 6.3 关卡页

**布局**：全屏沉浸，题目占满屏幕，背景是对应星球实时渲染。

**背景星球**：Canvas 实时渲染，随题目主题变化（选择题=冰蓝星球+光环，猜谜题=木星+大红斑，创意题=火星+尘埃环），星球持续旋转带大气光晕。

**顶部**：返回按钮 + 关卡标题（Orbitron字体）+ 积分徽章。

**步骤条**：三个圆形步骤点，完成变绿色✓，当前高亮紫色，逐题推进。

**科学家横幅**：每道题对应科学家头像 + 名言引用。

**题目卡片**：半透明深紫卡片，圆角20px，题型标签+题目+答题区。

**反馈强度（强烈）**：
- 答对：全屏"正确！"闪现 + 70颗彩色粒子爆炸 + 积分跳动放大
- 答错：全屏"再试试！" + 卡片左右抖动5次 + NOVA 给提示
- 创意题：全屏"CREATIVE!" + 粒子爆炸 + NOVA 个性化反馈

**NOVA气泡**：底部常驻，打字机逐字输出，随状态切换内容。

**通关界面**：连续5波粒子爆炸，展示积分+徽章+解锁科学家数。

---

## 6.4 NOVA 对话界面

**呈现方式**：全屏弹出，覆盖当前内容，scaleIn弹性动画（.35s）。

**触发方式**：任意页面右下角 NOVA 悬浮按钮（持续脉冲光晕）。

**头部**：动态表情头像（左）+ NOVA名称/角色/状态（中）+ 关闭按钮（右）。

**动态表情头像（Canvas实时绘制）**：
- 待机：微笑，偶尔眨眼，天线顶灯蓝紫色
- 思考：眼睛变橙色+旋转光环，3点跳动loading，持续900–1500ms
- 说话：嘴巴随文字节奏开合，眼睛变青绿色

**消息流**：
- NOVA消息：左对齐，深紫背景气泡，打字机输出（22ms/字）
- 用户消息：右对齐，紫色半透明气泡
- 思考动画：3点跳动，替换为回复后消失

**快捷问题**：4个预设太空问题按钮，点击直接发送。

**输入**：圆角输入框+发送按钮，Enter键发送，发送中禁用防重复。

---

## 6.5 科学家名人堂

**布局**：卡片墙，按三个阶段分组（Lv1/Lv2/Lv3），`auto-fill minmax(150px,1fr)` 自适应。

**名片卡片**：竖向比例（aspect-ratio .72）：
- 已解锁：紫色边框+光晕，hover上浮4px+放大
- 未解锁：灰度+锁头遮罩
- 最新解锁：右上角 NEW 橙色徽章

**详情卡**（点击后弹出）：
- rotateY翻转动画弹出，点击外区域关闭
- 内容：圆形头像 + 名字/年代/标签 + 名言引语（左侧紫线斜体）+ 生平简介 + 关联关卡标签 + 问NOVA快捷按钮

---

## 6.6 个人成就页

**定位**：平时查看的探索记录，与星渊证书分离。入口在导航栏。

**内容结构**：
- 用户信息卡（头像+姓名+学校+年级）
- 三格数据统计（星星积分 / 通关关卡 / 解锁科学家）
- 9个徽章格（已解锁发光，未解锁灰色剪影）
- 创意题答案回顾（按关卡分组，展示原始回答）
- "生成证书"按钮，跳转星渊证书页

---

## 6.7 星渊证书

**定位**：正式荣誉证书，截图分享用，从成就页进入。

**风格**：羊皮纸温暖调，烫金复古，Logo居中主导。

**证书结构（从上到下）**：
- 烫金外框（6px金色渐变描边，Canvas绘制四角装饰）
- 机构名：XING YUAN SPACE RESEARCH CENTER · 星渊宇宙研究中心
- 星渊Logo（96px星系漩涡金色SVG，视觉核心）
- 金色分隔线 + "荣誉证书 · 探索认证"（宋体）
- 兹证明 + **姓名**（32px深棕，最重要文字）
- 学校·年级
- 通关/科学家/积分描述文字
- 已获徽章行
- 底部三栏：颁发日期 ｜ 印章 ｜ 证书编号

**纸张质感**：Canvas生成羊皮纸噪点纹理+四周暗角晕染。

**印章渐显机制**（红色，-12°倾斜，右下角）：
- 1–2关：约14%透明度
- 3–4关：约30%透明度
- 5–6关：约52%透明度
- 7–8关：约74%透明度
- 9关全通：100%完整，触发淡入动画

**底部进度条**：金色渐变填充，显示印章完整度和剩余关卡数。

# 八、不做的功能

| 功能 | 原因 |
|------|------|
| 账号系统/云端存储 | 使用轻量本地登录+localStorage，无需后端账号体系 |
| 语音输入 | 浏览器兼容性问题多，不稳定 |
| 多人对战/排行榜 | 超出范围 |
| 积分兑换系统 | 不必要的复杂度 |
| 太阳系以外延伸天体单独详情页 | 关卡内已覆盖 |
